CDN（内容分发网络）部署在各大运营商的机房内。当用户通过浏览器请求资源时，可以直接反馈给用户，大大减轻了服务器数据中心的压力。本质上，CDN也是一种缓存。如果您的位置靠近CDN节点，那么网站响应速度也非常明显。另外，CDN缓存的资源主要是静态资源，比如静态页面、图片、CSS和JS文件。CDN加速对于一些覆盖范围广的网站是最有效的。比如阿里云CDN产品有280多个节点，运营商覆盖范围比较全面。
